麻雀×鬼ごっこ  

----------暫定ルール----------  
・全体ルール  
鬼 = 麻雀の親とする  
制限時間制で時間が経過したら1局終了　
時間内に誰かが上がったらその時点で局終了　
親は上がるか時間終了時に聴牌で鬼（親）続行  
麻雀と同じように得点で競う  

・逃走者  
逃げながらフィールドに落ちている牌を拾って役を完成させて上がる  

・鬼  
逃走者を追いながら牌を集めて上がる  

・追加事項  
牌に特殊能力をつける(例：東 -> 走力アップ、中 -> スタン)  

・現時点で未確定のルール  
鬼へどのようなアドバンテージを作るか  
逆転ができるようなルール  
麻雀のルールを崩さずに鬼ごっこに落とし込む
